## obZen

> Personal command center. Fully offline. Built for one.

A React + TypeScript PWA built entirely for my own life — no backend, no accounts, no cloud sync. All data lives in the browser's IndexedDB via Dexie.js. Installable on iOS and Android. Deployed to GitHub Pages.

**[→ Live App](https://prawnit279.github.io/obZen)** &nbsp;·&nbsp; **[Repository](https://github.com/Prawnit279/obZen)**

```
12 modules · 24 DB tables · 44 passing tests · Fully offline · PWA
```

| Module | What it does |
|---|---|
| Dashboard | Daily check-in, charts for volume, overload, macros, drum practice |
| Drum Studio | Practice timer, 39 rudiments, song library, PDF viewer, jam scheduler |
| Workout | Set-by-set session logging, 50+ exercise library, progressive overload |
| Yoga | 30-day tracker, 38 animated poses, breathing timer, sequence builder |
| Ayurveda | Pitta profile, dinacharya checklist, seasonal remedies |
| Calendar | Offline event management, 8 categories, CRUD |
| Projects | Kanban boards with drag-and-drop via @dnd-kit |
| Nutrition | Pitta-optimized macros, training/rest day toggle |
| Vedic Remedies | Planetary remedy log, mantra tracking |
| Meetings | Meeting log with agenda, outcomes, status |

**Stack:** React 18 · TypeScript (strict) · Vite · Tailwind CSS · Zustand · Dexie.js · Vitest · GitHub Pages CI/CD

**Design:** [Noircut](https://github.com/Prawnit279/obZen) — black/white, no gradients, no shadows, 1px borders, Inter font. 6 themes.

No analytics. No tracking. No subscriptions. Data never leaves the device.
